# sprint7-final-project-2
Análisis de Comportamiento de Consumo - ConnectaTel

## 🏢 Sobre la Empresa
**ConnectaTel** es una compañía de servicios móviles que busca optimizar su propuesta de valor a partir de los datos reales de uso de sus clientes.

## 🎯 Problema a Resolver
El objetivo principal fue transformar los datos brutos de consumo en un reporte estratégico que permitiera comprender cómo los clientes utilizan realmente los servicios móviles (voz, mensajes y datos). El propósito es identificar ineficiencias, mejorar la experiencia del usuario y ajustar la oferta comercial a las necesidades reales de cada segmento.

## 📊 Tipo de Análisis Realizado
Se llevó a cabo un análisis integral de datos enfocado en:
*   **Identificación de Patrones de Uso:** Detección de comportamientos de consumo fuera del promedio (outliers) y tendencias de uso intensivo.
*   **Segmentación de Usuarios:** Análisis comparativo por grupos demográficos para determinar si existen necesidades diferenciadas.
*   **Análisis de Rentabilidad y Upselling:** Detección de usuarios con alto consumo excedente en planes básicos para proponer migraciones a planes premium.
*   **Optimización Estratégica:** Desarrollo de recomendaciones basadas en datos para mejorar la fidelización y el Valor de Vida del Cliente (CLV).

## 💡 Valor Estratégico
El análisis permite a ConnectaTel pasar de una estrategia masiva a una **toma de decisiones basada en datos**, donde:
1. Se reducen los puntos de fricción (facturas impredecibles).
2. Se incrementa el ingreso recurrente mediante migraciones proactivas.
3. Se mejora la retención al ofrecer planes que realmente cubren el estilo de vida del usuario.
